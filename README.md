💰 Personal Finance Tracker

A dynamic web application that helps users manage their income and expenses with real-time balance updates and transaction tracking. Built using core web technologies, this project focuses on interactive UI and efficient data handling on the client side.

🚀 Features
💸 Add income and expense transactions
📊 Real-time balance calculation
📜 View complete transaction history
🏷️ Categorize transactions
✅ Form validation using JavaScript
🔄 Instant UI updates without page reload
💾 Data persistence using browser storage
🛠️ Tech Stack
Frontend: HTML5, CSS3, JavaScript
Data Handling: JSON
Storage: Browser Local Storage
